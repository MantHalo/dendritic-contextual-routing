# Dendritic Contextual Routing: Affine Modulation and Micro-Replay for Sequential Feature Conflicts

Public preprint draft v0, 2026-06-26.

Authors: MantHalo, OPAL-dev

## Abstract

Continual learning systems often receive task context as an auxiliary input, but some tasks require more than contextual concatenation: the same input features must be interpreted differently under different contexts. We introduce SDFC, a Same-Dimension Feature Conflict benchmark in which tasks share the same input dimensions while imposing antagonistic feature-label rules. The benchmark includes mirror task pairs that require context-dependent reinterpretation before a shared binary head. We compare FiLM-style affine modulation with a dendritic-inspired affine routing model that separates basal features from contextual modulation. In joint training, both contextual affine models solve the benchmark at approximately 96.3 percent final accuracy over 5 seeds. In sequential training without replay, both fall to approximately 64 percent final accuracy with approximately 43 percent forgetting. A replay buffer containing only 2 percent of each task's training set raises final accuracy to approximately 95.4 percent and reduces forgetting to approximately 1 percent. These results suggest that additive plus multiplicative contextual modulation is the relevant functional primitive in this setting, and that small replay buffers can preserve this solution under sequential feature conflict.

## Introduction

Context is often represented as an extra input. If the model receives the active task identifier, a sufficiently expressive network may learn to condition its prediction on that variable. Feature-conflict problems expose a stronger requirement. When the same observed dimensions carry different meanings across tasks, context must not merely be visible to the model. It must be able to transform the representation used by the predictor.

This work studies that requirement in a controlled setting. The goal is not to introduce a large benchmark, but to isolate a precise failure mode: the same feature space supports multiple incompatible task rules. In such a setting, a shared representation can be pulled in opposing directions, and a shared prediction head can only succeed if the hidden representation has already been transformed according to context.

The contributions are:

1. SDFC, a compact Same-Dimension Feature Conflict benchmark with explicit mirror task pairs.
2. A comparison between FiLM-style affine modulation and a dendritic-inspired separated affine routing model.
3. A sequential-learning replay sweep showing that 2 percent per-task replay nearly closes the observed gap to joint training.

The experiments support a narrow conclusion: in SDFC, the essential primitive is additive plus multiplicative contextual modulation of the hidden representation.

## SDFC Benchmark

SDFC asks whether a model can reinterpret the same input dimensions under different contexts. A fixed projection maps latent variables into a 32-dimensional input space. Each task receives inputs from this same space, but labels are generated by task-specific sign rules over latent dimensions.

In the four-task setup used for final results, two mirror conflicts are present:

- Task 0 and task 3 use opposite signs over the same two latent coordinates.
- Task 1 and task 2 form a second opposing pair.

The benchmark uses 4 tasks, 32 input dimensions, 8 latent dimensions, 10,000 training examples per task for final runs, 2,000 validation examples, 2,000 test examples, and a fixed benchmark seed of 12345 for the projection artifact.

## Method

The central computation studied here is affine contextual modulation:

```text
h = gamma(context) * h_basal + beta(context)
```

The basal pathway computes a hidden representation from the input. The contextual pathway computes multiplicative and additive modulation terms. The transformed representation is then passed to the predictor.

The `film_full` model implements FiLM-style affine modulation. The `dendritic_affine_separate` model separates basal computation from contextual modulation in a dendritic-inspired form:

```text
h = g(context) * h_basal + a(context)
```

The final experiments use a shared-head scenario, where task context must make a single binary logit meaningful for the active task.

Sequential experiments train tasks in order. Replay stores a fixed number of examples from each completed task. The final sweep evaluates 0 percent, 2 percent, 5 percent, and 10 percent replay, corresponding to 0, 200, 500, and 1000 examples per task.

## Results

Joint training confirms that contextual affine models can solve SDFC. The joint reference gives 96.28 percent mean final accuracy for `film_full` and 96.26 percent for `dendritic_affine_separate`, both over 5 seeds.

Without replay, sequential training degrades substantially:

| Model | Replay | Accuracy | Forgetting |
|---|---:|---:|---:|
| `film_full` | 0% | 63.91% | 43.16% |
| `dendritic_affine_separate` | 0% | 63.83% | 43.20% |

Task-level results show that the oldest mirror-conflicted task is heavily damaged. For `film_full`, task 0 ends at 27.98 percent final accuracy while task 3, the final task and its mirror antagonist, remains at 96.29 percent. For `dendritic_affine_separate`, task 0 ends at 28.35 percent and task 3 at 96.24 percent.

Adding a 2 percent replay buffer nearly closes the sequential gap:

| Model | Replay | Accuracy | Forgetting |
|---|---:|---:|---:|
| `film_full` | 2% | 95.44% | 1.05% |
| `dendritic_affine_separate` | 2% | 95.41% | 1.06% |

For `film_full`, task 0 recovers from 27.98 percent to 94.26 percent. For `dendritic_affine_separate`, task 0 recovers from 28.35 percent to 94.19 percent. Increasing replay beyond 2 percent provides smaller gains: both models reach about 95.9 percent with 5 percent replay and about 96.0 percent with 10 percent replay.

Across all replay budgets, the two affine contextual implementations remain nearly identical in final accuracy and forgetting. This supports functional equivalence in this SDFC regime. It does not establish a dendritic performance advantage over FiLM.

Gate-similarity diagnostics compare mirror and non-mirror task pairs. The measured cosine similarities remain high and change only modestly between 0 percent and 2 percent replay. The cautious interpretation is that replay stabilizes an already useful contextual solution rather than creating a wholly new routing organization.

## Discussion

SDFC isolates a specific representational demand. The model must use context to reinterpret shared features. In this setting, the result argues against treating context as a passive auxiliary input. The successful models transform the hidden representation through additive and multiplicative terms before prediction.

The replay results also show that representational capacity is not enough. Joint training can find a solution, but sequential training without replay damages it. Small replay buffers provide enough old-task evidence to prevent destructive reinterpretation of the same feature space.

The equivalence between FiLM and dendritic affine routing limits the claim to what the data support. A dendritic-inspired separated implementation can recover FiLM-like performance, but the benchmark does not establish superiority of the dendritic framing. Instead, it identifies a computational primitive that both models share.

## Relation To Contextual Recall

Recent Google Research work on ["Thinking to Recall"](https://research.google/blog/thinking-to-recall-how-reasoning-unlocks-parametric-knowledge-in-llms/) suggests that reasoning traces can unlock facts already present in model parameters through extra computation and factual priming. This is not direct evidence for SDFC, but it motivates a related future direction: latent access. A rule or representation may be present but difficult to retrieve without the right contextual cue, replay trace, or computation budget.

Future SDFC experiments could test neutral extra computation, prototype replay, verified contextual anchors, or pass@k-style probes to distinguish destroyed knowledge from inaccessible knowledge.

## Limitations

SDFC is synthetic and intentionally small. The final results use 4 tasks and a fixed projection. Additional work is needed to test more tasks, richer conflicts, alternative task orders, larger models, and non-synthetic data.

The gate-similarity diagnostics are descriptive and do not fully explain replay stabilization. The current replay sweep does not identify the exact minimum viable replay budget below 2 percent. The results should not be interpreted as evidence that replay is the only possible preservation mechanism.

## Conclusion

SDFC shows that when the same features carry conflicting meanings, context must modulate the representation used for prediction. FiLM-style affine modulation and a dendritic-inspired affine routing model both solve the benchmark under joint training and behave almost identically across replay budgets. Sequential training without replay damages the contextual solution, while a 2 percent per-task replay buffer nearly restores joint-level performance.
